خيارات التكامل مع التطبيقات الأخرى

1. تصدير الى ملف JavaScript منفصل

يمكنك تصدير الأسئلة إلى ملف JS منفصل واستيراده في تطبيقك:

ثم استخدم في تطبيقك:

<script src="questions.js"></script>

2. نسخ الأسئلة كـ JSON

يمكنك نسخ الأسئلة بتنسيق JSON واستخدامها في تطبيقك:

ثم استخدم في تطبيقك:

const questions = JSON.parse(questionsJson);

3. نسخ الأسئلة كـ JavaScript Object

يمكنك نسخ الأسئلة كقيمة JavaScript واستخدامها مباشرة في تطبيقك:

ثم الصق مباشرة في ملف JavaScript الخاص بك.

4. استخدام LocalStorage

الأسئلة مخزنة حاليًا في localStorage. يمكنك الوصول إليها من أي صفحة في نفس النطاق باستخدام:

const questions = JSON.parse(localStorage.getItem('questionData'));
